Création d'un alias
--
- Ce rendre dans le dossier utilisateur (exemple : /home/adrien/)
- Editer le fichier .bashrc
<pre><code>sudo kate .bashrc</code></pre>
- Ecrire l'alias à la fin du fichier : "alias cdad='cd /home/adrien/Labo/adrien.dhermy.fr'"
